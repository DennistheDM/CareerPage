# Dennis Makhandia — Career Page

This repository hosts a single-page portfolio for Dennis Makhandia, showcasing operations, strategy, and IT transformation experience. Open `index.html` to view the refreshed hero, impact cards, and enhanced navigation.

## Available files

- `dennis_makhandia_career_page.html` — the full branded page used for sharing.
- `index.html` — friendly entry point that redirects to the career page.
- `package.json` & `package-lock.json` — minimal Node metadata in case additional tooling is added later.
- `LICENSE` — MIT license governing reuse and distribution.

## Usage

1. Clone the repository.
2. Install dependencies (`npm install` or skip if no extra packages are required).
3. Open `index.html` (or simply `dennis_makhandia_career_page.html`) in a browser to view the page.
